# 📊 E-Commerce Project

## 🔍 Overview
This project presents a comprehensive Profit & Loss (P&L) analysis of an E-commerce business using Power BI. The dashboard provides insights into revenue, profitability, customer behavior, and operational efficiency.

The goal of this project is to transform raw data into meaningful business insights using interactive dashboards and data visualization techniques.

---

## 📌 Key Metrics
- 💰 Total Revenue: 49M
- 📦 Total Orders: 600
- 👥 Total Customers: 195
- 📊 Average Order Value: 81.23K
- 📈 Profit Margin: 35.35%
- 🚚 Average Delivery Time: 12 Days
- ❌ Cancellation Rate: 16.17%
- 🔁 Repeat Customers: 84.10%

---

## 📊 Dashboard Pages

### 1️⃣ Overview
- Revenue vs Orders trend (monthly)
- Order status breakdown (Delivered, Cancelled, Returned)
- KPI cards for quick business snapshot

---

### 2️⃣ Customer Analysis
- RFM Segmentation (Champions, Loyal, Potential, At Risk)
- New vs Repeat customer trends
- Customer distribution insights

---

### 3️⃣ Delivery Performance
- On-time vs delayed deliveries by region
- Order fulfillment analysis
- Regional delay visualization using map

---

### 4️⃣ Financial Insights (P&L)
- Profit margin analysis (35.35%)
- Sales vs Profit trends
- Category-wise loss and profit analysis
- Shipping cost impact on profitability

---

### 5️⃣ Customer Behavior
- Sales by category
- Average order value by category
- Monthly sales trends

---

### 6️⃣ Key Insights
- Strong profitability with 35.35% margin.
- High cancellation rate (16.17%) indicates operational inefficiencies.
- Average delivery time (12 days) is above industry expectations.
- Logistics and delivery delays impact customer satisfaction and retention.

---

## 🛠️ Tools & Technologies
- Power BI.
- DAX (Data Analysis Expressions).
- Power Query.
- Advanced Excel.

---

## 📈 Business Insights
- The business is profitable but operational inefficiencies exist.
- Delivery delays are a major factor affecting cancellations.
- High repeat customer rate (84%) shows strong customer retention.
- Reducing logistics cost and improving delivery speed can boost profitability.
